PrimeUI

Setup

Getting Started with PrimeUI is a straightforward process.

Download

PrimeUI is distributed in a zip file that is available at download page. Once you download the file and extract it in filesystem, you'd see two folders named development and production. Development folder has the uncompressed and the Production folder has the compressed distribution.

Include Resources

Although you can include resources of each widget individually, it is suggested to use the prime-ui.js and prime-ui.css file to register all the widgets at once. As PrimeUI is jQuery UI based, you also need to include jQuery and jQuery UI.

                                
                                    <link rel="stylesheet" href="%PATH%/jquery-ui.css" />
                                    <link rel="stylesheet" href="%PATH%/theme.css" />
                                    <link rel="stylesheet" href="%PATH%/prime-ui-min.css" />
                                    <script type="text/javascript" src="%PATH%/jquery.js"></script>
                                    <script type="text/javascript" src="%PATH%/jquery-ui.js"></script>
                                    <script type="text/javascript" src="%PATH%/prime-ui.js"></script>

Distribution also has a couple of images that are used by some of the widgets. The image folder should be placed at two level up for correct reference. For example if prime-ui.css is under %PATH%/resources/prime-ui/css/prime-ui-min.css location images folder should be at %PATH%/resources/images.
