Terminal

Terminal is a text based user interface. Backed in this demo is integrated with a restful webservice, try date and greet {param} commands.

<script type="text/javascript">
         $('#trmnl').puiterminal({
                    welcomeMessage:'Welcome to PrimeFaces Terminal, how are you today?',
                    handler: function(request, response) {
                        $.ajax({
                            type: "GET",
                            url: 'rest/terminal/' + request,
                            dataType: "text",
                            context: this,
                            success: function(data) {
                                response.call(this, data);
                            }
                        });
                    }
                });
</script>
