PrimeUI

PrimeUI is a collection of rich javascript widgets based on jQuery UI. PrimeUI is a spin-off from the popular JSF Component Suite, PrimeFaces.

Powered by HTML5

Built with HTML5, jQuery and CSS3, PrimeUI makes it trivial to create rich user interfaces in no time.

ThemeRoller Ready

PrimeUI Widgets are compatible with ThemeRoller. You can choose any predefined themes from ThemeRoller or special themes exclusive to PrimeUI.

jQuery and jQuery UI

PrimeUI is built with jQuery and jQuery UI APIs by following the same patterns and design principles.

PrimeFaces

PrimeUI is a part of PrimeFaces, extremely popular de-facto standard of UI Frameworks for Java Web Applications.
