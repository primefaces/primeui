Name Type Default Description
numVisible Integer 3 Number of visible items per page.
firstVisible Integer 0 Index of the first element to be displayed.
effectDuration Integer 500 Duration of the animation in milliseconds.
circular Boolean false Sets continuous scrolling.
responsive Boolean false Index of the first element to be displayed.
breakpoint Integer 400 Minimum width to switch to mobile mode (works only if responsive is enabled).

No events

Name Parameters Description
setPage index: Index of the page to switch to. Scrolls to the page with the given index.

Example

            $('#default').puicarousel('setPage', 1);
                                
            <script type="text/javascript">
                $(function() {
                    $('#default').puicarousel();

                    $('#custom').puicarousel({
                        numVisible: 2,
                        firstVisible: 1,
                        effectDuration: 1000,
                        circular: true,
                        responsive: true
                    });
                });
            </script>
