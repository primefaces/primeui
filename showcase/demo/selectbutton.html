Name Type Default Description
choices string,number null Choices for the buttons.

tabindex string 0 Order of elements in tabbing.

Name Parameters Description
change event: puiselectbuttonchange event
Fires when value changes.

            $('div').puiselectbutton({

Name Parameters Description
selectOption value Selects the given option.
unselectOption value Unelects the given option.

Example

            $('#single').puiselectbutton('selectOption',0);
                                
$(function() {
    $('#single').selectbutton({
                formfield: 'single_input',
                choices: [
                   {label:'Xbox One', value:0},
                   {label:'PS4', value:1},
                   {label:'Wii', value:2},
                   {label:'PS3',value:3}
                ]
            });
            
            $('#multiple').selectbutton({
                formfield: 'multiple_input',
                choices: [
                   {label:'Xbox One', value:0},
                   {label:'PS4', value:1},
                   {label:'Wii', value:2},
                   {label:'PS3',value:3}
                ],
                multiple: true
            });
});

Usage

SelectButton is similar to a select element that requires options as children. 
<p-selectbutton>
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
    <option value="3">Option 3</option>
    <option value="4">Option 4</option>
    <option value="5">Option 5</option>
</p-selectbutton>
