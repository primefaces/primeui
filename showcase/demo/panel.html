Panel Panel is a grouping component providing with content toggle and close features.

Name Type Default Description
title String/Custom Content null Header of the panel, custom content or string supported,defaults to title attribute of the element.
footer String/Custom Content null Footer of the panel, custom content or string supported.
toggleable Boolean false Defines if content of panel can be expanded and collapsed.
toggleDuration Number/String normal Defines the duration of the toggle effect, valid values are "slow", "normal", "fast" or a number in milliseconds.
toggleOrientation String vertical Defines orientation of toggle direction, valid values are "vertical" and "horizontal".
collapsed Boolean false Displays the panel as collapsed initially.
closable Boolean false Defines if content of panel can be hidden using close option.
closeDuration Number/String normal Defines the duration of the close effect, valid values are "slow", "normal", "fast" or a number in milliseconds.
enhanced boolean false In enhanced mode, panel dom is assumed to be existing and only events are bound. Default is false.
Name Parameters Description
beforeCollapse event: puipanelbeforecollapse event Fired before toggling.
afterCollapse event: puipanelaftercollapse event Fired after toggling.
beforeExpand event: puipanelbeforeexpand event Fired before toggling.
afterExpand event: puipanelafterexpand event Fired after toggling.
beforeClose event: puipanelbeforeClose event Fired before closing.
afterClose event: puipanelafterClose event Fired after closing.

Example

$('#default').puipanel({
    afterToggle: function(event) {
        //...
    }
});
Name Parameters Description
toggle - Toggles content.
expand - Expands content.
collapse - Collapses content.
close - Hides panel.
option name: Name of the option Returns the value of the option.
option name: Name of the option, value: Value of the option Set the value of the option.

Example

$('#default').puipanel('toggle');
                                
$('#default').puipanel();

$('#options').puipanel({
    toggleable: true,
    closable: true,
    title: function() {
        return '<i>Toggle and Close</i>';
    },
    footer: 'Footer Content'
});

Usage

Panel is used as a container element.

<p-panel title="PrimeElement" toggleable>
    The story begins as Don Vito Corleone, the head of a New York Mafia family, oversees his daughter's wedding.
    His beloved son Michael has just come home from the war, but does not intend to become part of his father's business.
    Through Michael's life the nature of the family business becomes clear. The business of the family is just like the head of the family,
    kind and benevolent to those who give respect,
    but given to ruthless violence whenever anything stands against the good of the family.
</p-panel>

Attributes

Name Type Default Description
id string null Identifier of the element.
title String null Text of header, defaults to title attribute of the element.
toggleable Boolean false Defines if content of panel can be expanded and collapsed.
toggleduration Number/String normal Defines the duration of the toggle effect, valid values are "slow", "normal", "fast" or a number in milliseconds.
togglerientation String vertical Defines orientation of toggle direction, valid values are "vertical" and "horizontal".
collapsed Boolean false Displays the panel as collapsed initially.
closable Boolean false Defines if content of panel can be hidden using close option.
closeDuration Number/String normal Defines the duration of the close effect, valid values are "slow", "normal", "fast" or a number in milliseconds.
onbeforecollapse function null Name of the function to invoke before collapse of content.
onaftercollapse function null Name of the function to invoke after collapse of content.
onbeforeexpand function null Name of the function to invoke before expansion of content.
onafterexpand function null Name of the function to invoke after expansion of content.
onbeforeclose function null Name of the function to invoke before panel is hidden.
onafterclose function null Name of the function to invoke after panel is hidden.

Methods

Name Parameters Description
toggle - Toggles content.
expand - Expands content.
collapse - Collapses content.
